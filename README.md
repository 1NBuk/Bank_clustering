# Проект: Customer Segmentation & Transaction Analysis для банка

## Цель:
Провести сегментацию клиентов банка на основе их транзакционной активности, демографии и баланса, чтобы выявить ключевые группы для таргетированного маркетинга и повышения лояльности.

## Описание проекта:

Проведен разведочный анализ данных (EDA), выявлены пропуски, дубликаты, выбросы и аномальные значения (например, отрицательный возраст клиентов).

Выполнена очистка данных:

заполнение пропусков (SimpleImputer),

корректировка некорректных значений (возраст клиентов),

обработка выбросов методом IQR.

Проведено feature engineering:

вычислен возраст клиента (CustomerAge),

закодированы категориальные переменные (CustGender, CustLocation),

стандартизированы числовые признаки (TransactionAmount, CustAccountBalance, TransactionTime).

Выполнена визуализация и анализ распределений: гистограммы, boxplots, heatmap корреляций.

Проведен кластерный анализ (KMeans) для сегментации клиентов:

определено оптимальное число кластеров методом локтя,

использован PCA для визуализации кластеров.

Проанализированы характеристики кластеров (средний баланс, сумма транзакций, возраст, распределение по полу и местоположению).

Выявлены VIP клиенты и группы с низкой активностью, сформулированы рекомендации для таргетированных маркетинговых кампаний.

Проведена сегментация клиентов по сумме транзакций и активности для определения стратегий стимулирования повторных покупок.

## Инструменты и технологии:

Язык: Python

Библиотеки: Pandas, NumPy, Matplotlib, Seaborn, scikit-learn (Pipeline, ColumnTransformer, SimpleImputer, StandardScaler, OneHotEncoder, KMeans, PCA)

Методы: EDA, feature engineering, обработка выбросов, кластеризация, PCA, визуализация данных

## Результаты и инсайты:

Идентифицировано 3 ключевых сегмента клиентов: массовый сегмент, слабововлеченные клиенты и VIP клиенты.

Для каждого сегмента предложены стратегии маркетинга и программы лояльности:

VIP клиенты → персонализированные премиальные предложения

Массовый сегмент → стандартные акции для удержания

Слабововлеченные → стимулирующие акции и кэшбэк

Построена визуализация распределения транзакций и кластеров для наглядной демонстрации сегментации.

## Эффект:

Сегментация позволяет банку оптимизировать маркетинговый бюджет, направляя усилия на ценные сегменты клиентов.

Повышение вероятности удержания клиентов и увеличения среднего чека за счет таргетированных предложений.
